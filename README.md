Educational layout project built from a Figma design during frontend learning.
The project focuses on responsive layout, integration of JavaScript plugins, accessibility, and cross-browser compatibility.

## Tech Stack
- HTML5 (semantic markup)
- CSS3
- JavaScript (Vanilla JS)
- Flexbox & CSS Grid
- BEM methodology

## Features
- Fully responsive layout (desktop / tablet / mobile according to the design)
- Slider implemented using a JavaScript plugin
- Accordion component implemented using a JavaScript plugin
- Interactive UI elements (hover, active, focus states)
- Smooth transitions and animations
- Mobile-friendly navigation

## Accessibility
- Keyboard navigation support
- Visible focus states for interactive elements
- Semantic HTML structure
- ARIA attributes for interactive components where needed

## Cross-Browser Compatibility
Tested in modern browsers:
- Chrome
- Firefox
- Edge

## Getting Started
To run the project locally:
1. Clone the repository
2. Open `index.html` in your browser
